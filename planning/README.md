# Planning Manifests

The completed prose planning corpus for the agents platform overhaul has been
removed from the active tree after archive completion. Historical planning
evidence remains available in git history and in the archived OpenSpec changes
under `openspec/changes/archive/2026-05-01-*`.

This directory now keeps only active machine-readable planning ledgers under
`planning/manifests/`. Treat those files as conservative support evidence, not
as completed implementation or validation proof.

Use the durable OpenSpec specs under `openspec/specs/` for current requirements.
Create a new OpenSpec change for future planning work instead of reviving the
removed planning docs.
