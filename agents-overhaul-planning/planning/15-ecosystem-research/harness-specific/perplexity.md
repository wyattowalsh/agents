# Perplexity Desktop Ecosystem Surface

## Sources

- Local/remote MCP help: https://www.perplexity.ai/help-center/en/articles/11502712-local-and-remote-mcps-for-perplexity
- Perplexity MCP server: https://docs.perplexity.ai/guides/mcp-server

## Extension surfaces

- Local MCP on macOS Mac App Store app.
- Helper app `PerplexityXPC` required for local MCP.
- Remote MCP announced as coming soon.
- Perplexity MCP server for search/reasoning via MCP.

## Planning implications

- Treat Perplexity Desktop support as experimental unless import/export config contract is verified.
- Prefer docs/setup guide over automatic writes.
- Perplexity MCP server is useful for live web/reasoning, not a skills replacement.
