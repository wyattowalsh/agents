# Harness-Specific Ecosystem Research

Each file captures idiosyncratic extension surfaces for a target AI tool. These are ingredients for registry/adapters/task graph updates, not claims of complete implementation.

## Required sections per harness

- Official/primary sources.
- Extension surfaces.
- Skill compatibility.
- MCP compatibility.
- Config scoping.
- Automation affordances.
- Security and stability caveats.
- UI/UX patterns to reuse.
- Planning implications.
