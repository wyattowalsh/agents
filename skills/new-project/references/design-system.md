# Design System Baseline

Generate a lean `DESIGN.md`, not a bloated component platform.

Include:

- product purpose and audience
- visual stance
- mobile-first navigation and density rules
- typography scale and line-length constraints
- one accent palette and neutral scale
- spacing rhythm
- motion constraints and reduced-motion requirements
- accessibility constraints
- anti-slop checklist

Avoid generic gradients, unused bento grids, gratuitous glassmorphism, excessive animation, and unused shadcn components unless grounded in project context.
