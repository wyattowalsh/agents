## 6 Naming Archetypes

### 1. Invented Words

Phonetically constructed neologisms with no prior meaning. Use when trademark defensibility and availability matter most.

**Technique:** Combine high-impact consonants with vowels matching the emotional register. Target 2-3 syllables. Test by saying each candidate aloud 5 times rapidly.

| Name | Why It Works |
|------|-------------|
| Kodak | Hard K bookends = snap/click; short, symmetric |
| Xerox | X-opening = innovation; -ox = scientific precision |
| Hulu | Soft double-U = warmth; cross-linguistic ease |
| Roku | R + hard K = energy; two syllables, first-stress |
| Skype | SK- = swift; single syllable; "type" rhyme |
| Venmo | V = vibrant; -mo = casual/friendly |
| Twilio | Tw- = connection; -ilio = Italian elegance |
| Zillow | Z = attention; -illow echoes "pillow" (comfort) |
| Brex | BR- plosive = strength; X-ending = edge |
| Quora | Qu- = questioning; -ora = "oracle" echo |

**Pitfalls:** Unpronounceable clusters; too similar to existing brands; so meaningless it's forgettable; overuse of X/Z/Q.

### 2. Metaphorical Transfers

Concrete concepts from other domains evoking product qualities. Use for emotional resonance and narrative potential.

**Technique:** List 5-10 product qualities. For each, brainstorm objects, natural phenomena, or mythology that embody it. Cross-pollinate: nature, geography, music, physics.

| Name | Source | Transferred Meaning |
|------|--------|---------------------|
| Amazon | Geography | Vast, everything flows through it |
| Slack | Sailing | Ease, flexibility, breathing room |
| Rust | Chemistry | Systems-level, raw, close to metal |
| Spark | Physics | Ignition, catalyst |
| Glacier | Nature | Massive storage, persistence |
| Apex | Geography | Summit, top-of-category |
| Obsidian | Geology | Strong, sharp; forged under pressure |
| Harbor | Maritime | Safe haven, orchestration hub |
| Kindle | Fire | Ignite, warmth, starting something |
| Eclipse | Astronomy | Transformation, overwhelming power |

**Pitfalls:** Too obscure (needs explanation = failed); negative secondary meanings; already claimed in category; cultural specificity.

### 3. Compound Blends

Portmanteaus or morpheme combinations. Use when clarity of purpose matters and two concepts tell the story.

**Technique:** List key concept pairs (action+object, quality+domain). Try: full compound (YouTube), blended overlap (Instagram), truncated compound (Pinterest), phonetic merge.

| Name | Components | Technique |
|------|-----------|-----------|
| Instagram | instant + telegram | Truncated blend |
| YouTube | you + tube | Full compound |
| Pinterest | pin + interest | Overlap blend |
| Cloudflare | cloud + flare | Full compound |
| Datadog | data + dog | Full compound (watchdog metaphor) |
| MailChimp | mail + chimp | Full compound (playful) |
| AirTable | air + table | Full compound (lightweight + database) |
| Postman | post + man | Full compound (mail carrier) |
| Buildkite | build + kite | Full compound (lightweight CI) |
| Snapchat | snap + chat | Full compound (action pair) |

**Pitfalls:** Awkward phonetic seams; exceeding 10 chars kills memorability; forced overlap; both components too generic.

### 4. Classical Roots

Latin/Greek etymology for gravitas and cross-linguistic recognition. Use for enterprise, scientific, or premium positioning.

**Technique:** Find Latin/Greek equivalents of core concept. Modify for modern pronunciation. Add suffixes (-a, -o, -is). Verify root meaning is real.

| Name | Root | Origin | Meaning |
|------|------|--------|---------|
| Nike | Nike | Greek | Goddess of victory |
| Lumen | Lumen | Latin | Light |
| Nova | Nova | Latin | New; exploding star |
| Nexus | Nexus | Latin | Connection, bond |
| Flux | Fluxus | Latin | Flow, change |
| Vox | Vox | Latin | Voice |
| Prisma | Prisma | Greek | Light separation |
| Vercel | Accelerare | Latin | Speed (truncated) |
| Coda | Coda | Latin | Concluding passage |
| Astra | Astrum | Latin | Star(s) |

**Pitfalls:** Pseudo-Latin gibberish; negative Romance-language meanings; pretentious tone mismatch; overcrowded namespace.

### 5. Evocative Fragments

Short, punchy, abstract names that feel meaningful without being literal. Use for developer tools and creative brands.

**Technique:** Strong 3-5 letter combinations. Single or two syllables, first-stress. Borrow from short imagery words, then modify: drop a letter, swap a vowel.

| Name | Why It Works |
|------|-------------|
| Figma | "Figure" echo; 5 letters, punchy |
| Neon | Bright, electric; 4 letters, strong imagery |
| Zed | Terminal, sharp; 3 letters, Z-impact |
| Deno | "Node" anagram; fresh start |
| Bun | Tiny, fast, contained; 3 letters |
| Warp | Speed, distortion, sci-fi; 4 letters |
| Arc | Geometry, trajectory; 3 letters |
| Fly | Movement, lightness; 3 letters |
| Plex | "Multiplex" echo; 4 letters |
| Sumo | Strength, mass; 4 letters |

**Pitfalls:** Too short = SEO nightmare; common English word = domain taken; sounds like another product aloud.

### 6. Descriptive-Creative

Clearly communicates function with linguistic flair. Use in crowded markets where the audience needs immediate category recognition.

**Technique:** Start with the literal function. Find one evocative modifier. Combine. Test: can someone guess the category from the name alone?

| Name | Why It Works |
|------|-------------|
| Cloudflare | Cloud infra + flare (protection signal) |
| Fastly | Speed as identity; adverb form feels active |
| Tailwind | CSS utility = "wind at your back" |
| Sentry | Error monitoring = standing guard |
| PlanetScale | Database scaling = planetary ambition |
| Retool | Re- prefix tells the rebuild story |
| LaunchDarkly | Feature flags = dark launches |
| Mintlify | Fresh docs + action suffix |
| Grafana | Graph + -ana (analysis feel) |
| Snyk | "Sneak" respelling (security scanning) |

**Pitfalls:** Too literal = generic; description locks you into current scope; meaningless modifier (SmartX, ProY).

## Sound Symbolism Guide

Research-backed associations (Lexicon Branding, Klink 2000, Yorkston & Menon 2004).

### Consonant Effects

| Class | Letters | Perception | Best For |
|-------|---------|-----------|----------|
| Plosives | B, P, K, T, D, G | Energy, impact, boldness | Dev tools, performance |
| Sonorants | L, M, N, R | Warmth, flow, smoothness | Consumer, creative |
| Fricatives | F, S, Sh, V, Z | Speed, elegance, precision | Design, fintech |
| Affricates | Ch, J, Dg | Playfulness, informality | Games, social |

**Key individual consonants:** V=vibrant, B=reliable, Z=attention, X=innovation, K=crisp, S=sleek, R=robust, N=natural.

### Vowel Effects

| Vowel | Class | Association | Examples |
|-------|-------|-------------|---------|
| ee, ih | Front high | Small, fast, precise | Figma, Vite, Stripe |
| eh, ae | Front mid/low | Sharp, energetic | Redis, Slack |
| ah, aw | Back low | Large, powerful | Docker, Kafka |
| oo, uh | Back high | Smooth, cool, deep | Hulu, Roku, Sumo |
| oh | Back mid | Open, stable, warm | Notion, Coda |

### Stress Patterns

| Pattern | Effect | Examples |
|---------|--------|----------|
| First-syllable (trochee) | Confident, easiest to remember | **FIG**-ma, **STRI**-pe |
| Second-syllable (iamb) | Elegant, formal | Ver-**CEL** |
| Equal (spondee) | Balanced, solid | **CLOUD**-**FLARE** |
| Single syllable | Maximum punch | Bun, Arc, Warp, Zed |

**Sweet spot:** Two syllables, first-syllable stress.

## Phonetic Pattern Library

| Pattern | Structure | Examples | Best For |
|---------|-----------|----------|----------|
| CVC | consonant-vowel-consonant | Bun, Warp, Arc, Zed | CLI (max density) |
| CVCV | C-V-C-V | Roku, Hulu, Nike, Vite | Modern, open feel |
| CVCCV | C-V-C-C-V | Vercel, Fastly, Sentry | Enterprise weight |
| CVCVC | C-V-C-V-C | Figma, Nikon | Complete, balanced |
| VCV | V-C-V | Uber, Okta | Approachable |
| CVC+CVC | compound | Postman, Datadog | Clear compound names |

**By context:** CLI = CVC/CVCV (max 6 chars). OSS = CVCV/CVCVC (max 8). Product = any (max 10). Brand = any (max 12).

## Context-Specific Strategies

### CLI Tools
- Max 6 chars; CVC/CVCV patterns; lowercase only; no hyphens
- Evocative fragments and invented words win (short + unique = available)
- Test `which <name>` for system command collisions
- Every character costs typing friction; optimize ruthlessly

### OSS Libraries
- Must be a valid identifier in target language; 4-8 chars ideal
- Descriptive-creative archetype works well (guess the purpose from the name)
- Check registry availability FIRST; avoid version-sounding names (v2, modern)

### Products (SaaS, Apps)
- .com domain still king; two syllables, first-stress optimal
- Metaphorical transfers create marketing narratives
- Test morphological flexibility: "Just Slack it to me"
- International pronunciation matters for global products

### Brands
- Classical roots add gravitas; sonorants for warmth, plosives for authority
- Handle consistency across 5+ platforms is mandatory
- Avoid trends (-ly, AI-, -GPT) that age the name

### Creative Projects
- Memorability above all; affricates (Ch, J) feel playful
- Compound blends with imagery stick (MailChimp, Buildkite)
- Longer names acceptable (up to 12 chars); test hashtag uniqueness

### Side Projects
- Availability above quality; .dev is fine if .com is taken
- Invented words maximize availability with minimal effort
- Reserve 10 minutes max for naming; diminishing returns
