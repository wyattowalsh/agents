# Synthesis

Synthesis is not a summary of who said what. It is the panel's intellectual product.

## Required Components

- Identify the underlying axiom that explains the disagreement.
- State the emergent question that none of the panelists started with.
- Identify what evidence would resolve the remaining tension.
- Map the positions structurally, not as a list of opinions.
- Name the uncomfortable implications that surfaced.
- Give 3-5 key takeaways.
- Provide genuine further reading, using real works only.
- Self-assess whether the panel produced real insight beyond a single expert.

## Follow-Up Options

After synthesis, generate 3-4 numbered follow-up options.

- Each option must reference a specific tension, expert, or emergent question.
- Do not use generic options like "ask more questions."
- If the user is making a practical decision, connect the synthesis to that choice explicitly.
