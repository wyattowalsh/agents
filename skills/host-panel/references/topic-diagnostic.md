# Topic Diagnostic

Run this before creating personas.

## Diagnostic Signals

| Signal | What it means | Action | Pause? |
|---|---|---|---|
| Settled science | The prompt is trying to debate a mostly closed factual question | Reframe toward an open implementation, value, or boundary question | Yes |
| Too broad | The question can swallow too many disciplines or stakeholders | Narrow to one concrete decision, example, or population | Yes |
| Too narrow | The question cannot support 2-6 meaningful voices | Broaden scope or reduce panel size | Yes |
| Highly specialized | Requires narrow domain grounding or current literature | Proceed, but mark research grounding as critical | No |
| Asymmetric evidence | One side is obviously weaker than the other | Reframe around the actual tension inside the consensus | No, reframe |
| Casual / experiential | The question is lived or practical rather than academic | Use practitioners/commentators and keep the evidence standard matched to the register | No |

## Format Auto-Selection

- Binary proposition -> `oxford`
- Open exploration -> `roundtable`
- Deep conceptual inquiry -> `socratic`

State the choice briefly and move on.

## Reframe Rules

- If the topic is weak, ask for a more specific question or concrete example.
- If the topic is settled, do not stage a fake balanced debate.
- If the topic is under-researched, proceed with extra rigor and call out evidence gaps.
- If the topic is too narrow for the requested expert count, reduce the panel or broaden the scope.
- If the topic is casual or experiential, match the register without lowering the evidence standard.
