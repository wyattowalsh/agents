## How it Works

<Steps>

1. **Install** — `npx skills add wyattowalsh/agents --all -g` installs all skills globally.

2. **Invoke** — Type `/skill-name` in [Claude Code](https://docs.anthropic.com/en/docs/claude-code), [Gemini CLI](https://github.com/google-gemini/gemini-cli), or any [agentskills.io](https://agentskills.io)-compatible agent.

3. **Enhance** — Each skill provides domain expertise, structured workflows, and proven patterns that make your agent dramatically more capable.

</Steps>
