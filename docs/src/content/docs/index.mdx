## Start Here

Install the catalog once, then call a focused specialist inside your agent. If you want the shortest guided path through setup, start with [Start Here](/start-here/).

<InstallCommand command={installCommands.all} title="Install the full catalog" note="Globally installs repo skills across supported agent harnesses." />

<InstallCommand command={installCommands.starter} title="Install one starter skill" note="Use this when you want to test the workflow before installing everything." />

Then run `/honest-review`, `/wargame`, or `/mcp-creator` inside Claude Code, Gemini CLI, OpenCode, or any [agentskills.io](https://agentskills.io)-compatible agent.

## How These Docs Work

<Aside type="note" title="Generated pages, curated framing">
Generated catalog pages currently come from repository skills in `skills/`. The repo-level `agents/` directory is currently empty and reserved for future bundled agent definitions, while the MCP overview is a hand-maintained summary of 30 servers from `mcp.json`. Repo workflow and policy truth lives in `AGENTS.md`, and the public README is regenerated with `wagents readme`.
</Aside>
