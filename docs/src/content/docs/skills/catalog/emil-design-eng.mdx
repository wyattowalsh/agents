## Purpose

Design engineering skill distilled from Emil Kowalski (animations.dev) articles and course. Encodes philosophy on UI polish, component design, animation decisions (timing &lt;300ms, custom easing), perceived performance, invisible details that make interfaces feel great. Uses before/after tables and concrete rules for reviews.
