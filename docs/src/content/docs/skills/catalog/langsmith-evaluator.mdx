### Quick Start

**Install:**
```bash
npx skills add langchain-ai/langsmith-skills --skill langsmith-evaluator --skill langsmith-dataset --skill langsmith-trace -y -g -a antigravity claude-code codex crush cursor gemini-cli github-copilot grok opencode
```

**Use:** `/langsmith-evaluator`
