# External Skills

This page tracks the curated third-party Agent Skills selected for this repo's
skill expansion. External skills are trust-bearing assets, not inert docs:
inspect `SKILL.md`, hooks, scripts, commands, network calls, credential
handling, and source provenance before repo promotion.

<Aside type="caution">
Do not bulk-import entire external repositories. Install only the curated
skills listed here, and run `external-skill-auditor` before repo promotion.
</Aside>

Target agents:

- `antigravity`
- `claude-code`
- `codex`
- `gemini-cli`
- `github-copilot`
- `opencode`

## Keep Global Only Or Avoid

- `vercel-labs/skills@find-skills`: duplicate of `discover-skills`.
- `vercel-labs/agent-browser`: broad operational scope; overlaps browser automation.
- `anthropics/skills`: inspect case-by-case because many entries duplicate local or system skills.
- `huggingface/skills`: keep global unless ML workflows become frequent.
- Community Better Auth, Clerk, Neon, and generic OpenAI Agent SDK skills: lower source confidence or duplicate architecture coverage.
- `docs.stripe.com` registry entries: avoid until source syntax and provenance are resolved.
