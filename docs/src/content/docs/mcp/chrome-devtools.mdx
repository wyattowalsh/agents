## Registry Connection

Representative stdio launch block from `config/mcp-registry.json`:

```json
{
  "mcpServers": {
    "chrome-devtools": {
      "command": "bash",
      "args": [
        "${REPO_ROOT}/scripts/mcphub/chrome-devtools-browser-url.sh"
      ],
      "env": {
        "CHROME_DEVTOOLS_MCP_NO_USAGE_STATISTICS": "1",
        "CHROME_DEVTOOLS_MCP_NO_UPDATE_CHECKS": "1"
      }
    }
  }
}
```
