## Registry Connection

Representative stdio launch block from `config/mcp-registry.json`:

```json
{
  "mcpServers": {
    "ossinsight": {
      "command": "npx",
      "args": [
        "-y",
        "ossinsight-mcp"
      ]
    }
  }
}
```
