## Registry Connection

Representative stdio launch block from `config/mcp-registry.json`:

```json
{
  "mcpServers": {
    "fetcher": {
      "command": "npx",
      "args": [
        "-y",
        "fetcher-mcp"
      ]
    }
  }
}
```

| Field | Value |

| enabled | `True` |
