## Registry Connection

Representative stdio launch block from `config/mcp-registry.json`:

```json
{
  "mcpServers": {
    "penpot": {
      "command": "npx",
      "args": [
        "-y",
        "mcp-remote",
        "https://design.penpot.app/mcp/stream?userToken=${PENPOT_MCP_USER_TOKEN}"
      ],
      "env": {
        "PENPOT_MCP_USER_TOKEN": "${PENPOT_MCP_USER_TOKEN}"
      }
    }
  }
}
```

## Registry Entry

| Field | Value |
| ----- | ----- |
| id | `penpot` |
| transport | `stdio` |
| command | `npx` |
| enabled | `True` |
