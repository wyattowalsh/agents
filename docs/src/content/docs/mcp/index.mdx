<Aside type="tip" title="What is MCP?">
The [Model Context Protocol](https://modelcontextprotocol.io/) is an open standard for connecting AI agents to external tools and data sources. Each server below exposes one or more tools that agents can invoke during a session.
</Aside>

<Aside type="note" title="Source of truth for this page">
This page is **hand-maintained** and currently summarizes the MCP server entries in the repository root `mcp.json`. That file includes a mix of local scripts, local packages, remote gateways, and third-party servers, so this list is broader than the repo-local `mcp/` directory.
</Aside>

## Managed client sync coverage

<Aside type="tip" title="Sync source of truth">
Edit `config/mcp-registry.json`, then preview local drift with `uv run python scripts/sync_agent_stack.py --targets home --check`. Apply repo and home updates with `uv run python scripts/sync_agent_stack.py --targets repo,home --apply`.

## DuckDuckGo setup (representative client snippets)

Use server key `duckduckgo-search` with the `duckduckgo-mcp-server` package.

#### Standard JSON clients (`mcpServers`)

Applies to ChatGPT Desktop, Claude Desktop, Cursor, Gemini CLI, Gemini antigravity, and the Gemini extension config.

```json
{
  "mcpServers": {
    "duckduckgo-search": {
      "command": "uvx",
      "args": ["duckduckgo-mcp-server"]
    }
  }
}
```

#### Codex (`~/.codex/config.toml`)

```toml
[mcp_servers.duckduckgo-search]
command = "uvx"
args = ["duckduckgo-mcp-server"]
```

#### GitHub Copilot CLI (`~/.copilot/mcp-config.json`)

```json
{
  "mcpServers": {
    "duckduckgo-search": {
      "type": "stdio",
      "command": "uvx",
      "args": ["duckduckgo-mcp-server"],
      "tools": ["*"]
    }
  }
}
```

#### OpenCode (`~/.config/opencode/opencode.json`)

```json
{
  "mcp": {
    "duckduckgo-search": {
      "type": "local",
      "command": ["uvx", "duckduckgo-mcp-server"],
      "enabled": true
    }
  }
}
```

#### AITK (`~/.aitk/mcp.json`)

```json
{
  "servers": {
    "duckduckgo-search": {
      "type": "stdio",
      "command": "uvx",
      "args": ["duckduckgo-mcp-server"]
    }
  }
}
```

## Chrome DevTools setup (representative client snippets)

Use server key `chrome-devtools` with `chrome-devtools-mcp@latest`. The managed baseline passes `--autoConnect`.

#### Standard JSON clients (`mcpServers`)

Applies to ChatGPT Desktop, Claude Desktop, Cursor, Gemini CLI, Gemini antigravity, and the Gemini extension config.

```json
{
  "mcpServers": {
    "chrome-devtools": {
      "command": "npx",
      "args": ["-y", "chrome-devtools-mcp@latest", "--autoConnect"]
    }
  }
}
```

#### Codex (`~/.codex/config.toml`)

```toml
[mcp_servers.chrome-devtools]
command = "npx"
args = ["-y", "chrome-devtools-mcp@latest", "--autoConnect"]
```

#### GitHub Copilot CLI (`~/.copilot/mcp-config.json`)

```json
{
  "mcpServers": {
    "chrome-devtools": {
      "type": "stdio",
      "command": "npx",
      "args": ["-y", "chrome-devtools-mcp@latest", "--autoConnect"],
      "tools": ["*"]
    }
  }
}
```

#### OpenCode (`~/.config/opencode/opencode.json`)

```json
{
  "mcp": {
    "chrome-devtools": {
      "type": "local",
      "command": ["npx", "-y", "chrome-devtools-mcp@latest", "--autoConnect"],
```
