<Aside type="note" title="Registry MCP server (hand-maintained)">

Do not run `wagents docs generate` on this file (HAND-MAINTAINED sentinel).

<Aside type="note" title="MCPHub routing">
When MCPHub is enabled, clients can reach this server at 
`http://127.0.0.1:46683/mcp/duckduckgo-search` via the local control plane. 
See [MCPHub](/mcp/mcphub/) for the repo-owned metadata server and launch details.
</Aside>

<Aside type="caution" title="Client key alias">
Registry and MCPHub use server key `duckduckgo-search`, while several client examples on [MCP Overview](/mcp/) use the shorter key `duckduckgo` with the same `uvx duckduckgo-mcp-server` launch command.
</Aside>


## Registry Connection

Representative stdio launch block from `config/mcp-registry.json`:

```json
{
  "mcpServers": {
    "duckduckgo-search": {
      "command": "uvx",
      "args": [
        "duckduckgo-mcp-server"
      ]
    }
  }
}
```

## Registry Entry

| Field | Value |
