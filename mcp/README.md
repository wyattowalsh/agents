# MCP

First-party MCP servers authored in this repository live in `mcp/<name>/` (tracked).

- `mcphub/` — MCPHub control plane (tracked first-party server)

Machine-local untracked dirs (per `.gitignore`):

- `archives/`
- `cache/`
- `notes/`
- `secrets/`
- `servers/`

Only the scaffold (`.gitignore`, `README.md`) and first-party `mcp/<name>/` trees are tracked. Local installs and assets stay untracked.
