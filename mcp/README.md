# Local MCP Workspace

This directory is the machine-local MCP workspace for this checkout.

- `servers/` contains third-party MCP server checkouts and build artifacts.
- `archives/` contains downloaded archives.
- `secrets/` contains local credential or config files.
- `notes/` contains local notes, indexes, and scratch files.
- `cache/` contains generated caches and temporary state.

Only this scaffold is tracked. The actual MCP installs and local assets stay untracked.
